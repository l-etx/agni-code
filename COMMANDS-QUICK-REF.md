# Commands Quick Reference

> 19 slash commands. Type `/` in any Claude Code session to invoke.

---

## Planning

| Command | What it does |
|---------|-------------|
| `/plan` | Restate requirements and create step-by-step implementation plan |
| `/plan-prd` | Generate a lean PRD then hand off to `/plan` |

## Code Review

| Command | What it does |
|---------|-------------|
| `/code-review` | General code quality review |
| `/python-review` | Python-specific code review |
| `/cpp-review` | C++ code review |
| `/fastapi-review` | FastAPI endpoint review |
| `/review-pr` | Review a pull request |

## Security

| Command | What it does |
|---------|-------------|
| `/security-scan` | Run security scan on current codebase |

## Refactoring & Docs

| Command | What it does |
|---------|-------------|
| `/refactor-clean` | Safe refactoring and dead code removal |
| `/update-docs` | Update documentation |
| `/pr` | Pull request workflow |
